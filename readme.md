# THEMEABLE REACT NATIVE

![Rect native themes](https://i.imgur.com/67s5nNh.png)

### Prerequisites

- npm
- node js
- react native

### Installing

clone the repo

```
git clone https://github.com/muneneevans/themeable-react-native.git
```

install node modules

```
npm install
```

## Running the project

```
react-native run-android
```
